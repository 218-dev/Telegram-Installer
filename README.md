# Telegram IPA Installer

موقع تثبيت Telegram IPA مع ميزات إضافية

## المميزات
- واجهة مستخدم جميلة وعصرية
- تأثيرات بصرية جذابة
- سهولة التثبيت بنقرة واحدة
- متوافق مع أجهزة iOS

## كيفية الاستخدام
1. قم برفع الملفات على GitHub Pages
2. استبدل رابط الـ IPA برابط ملفك
3. تأكد من صحة bundle identifier في ملف plist

## المتطلبات
- استضافة GitHub Pages
- ملف IPA صالح للتثبيت
